# Phase1 Product Management API

A Spring Boot RESTful API for managing products. This project demonstrates a clean architecture using **DTOs**, **service layer**, **exception handling**, and **pagination**.

## Features

- Create, Read, Update, Delete (CRUD) operations for products
- DTO-based responses for clean API output
- Centralized exception handling with `@ControllerAdvice`
- Pagination support for listing products
- Built with Spring Boot, Spring Data JPA, and MySQL

## Technologies Used

- Java 17
- Spring Boot 4.x
- Spring Data JPA / Hibernate
- MySQL
- Maven

## Getting Started

### Prerequisites

- Java 17+
- Maven
- MySQL database
